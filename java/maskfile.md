# Calculator - Java Swing Application

> A simple Swing GUI Calculator application

## build

> Compiles the Java source files

```bash
mkdir -p out
javac -d out src/main/java/com/example/calculator/Main.java
echo "Build complete!"
```

## run

> Compiles and runs the application

```bash
mkdir -p out
javac -d out src/main/java/com/example/calculator/Main.java
java -cp out com.example.calculator.Main
```

## clean

> Removes compiled class files

```bash
rm -rf out
echo "Clean complete!"
```
